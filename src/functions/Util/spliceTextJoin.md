`$spliceTextJoin` 

## Usage

```php
$spliceTextJoin[text;splitter;sep1?;sep2?]
```

## Parameters 


| Field    | Type   | Description | Required |
| -------- | ------ | ----------- | -------- |
| text     | string |             | yes      |
| splitter | string |             | yes      |
| sep1     | string |             | no       |
| sep2     | string |             | no       |
